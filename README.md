
# Angular | .NET Test

---

To view the project, start both api and ui

## Start API

Use commands below to start the api

```
 	cd api
	dotnet run build
	dotnet run
```

## Start UI

Use commands below to start the api

```
	cd ui
	npm install
	ng serve -o
```
This will open the UI in the browser using link `http://localhost:4200/`
